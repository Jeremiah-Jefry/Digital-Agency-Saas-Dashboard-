# VEXORIUM — Futuristic Agency OS

Production-ready frontend SaaS experience rebuilt with:

- Next.js App Router
- Tailwind CSS
- Framer Motion
- shadcn-style reusable UI primitives
- Lucide React
- Recharts
- localStorage-powered frontend interactions

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Routes

- `/` marketing landing page
- `/dashboard`
- `/clients`
- `/billing`
- `/team`
- `/settings`
- `/login`
- `/signup`
- `/forgot-password`
- `/otp-verification`

## Architecture

- `app/` route-level pages and layouts
- `components/` reusable UI, dashboard, marketing, auth, chart, and layout modules
- `data/` mock SaaS data sources
- `hooks/` localStorage state utilities
- `lib/` shared utilities
